# For users

## How to install
```bash
pip3 install https://github.com/Kalebe16/data-fox.git
```

## How to run
```bash
data-fox
# Or
python3 -m data_fox
```

# For developers

## How to lint
```bash
blue ./data_fox; isort ./data_fox
```
